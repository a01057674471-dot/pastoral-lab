# Pastoral Lab 반응형 V2

PC, 태블릿, 모바일에서 화면 구성이 자동으로 바뀌는 반응형 버전입니다.

## 화면별 구성
- PC: 좌측 고정 메뉴 + 넓은 작업 영역 + 2열 홈 카드
- 태블릿: 상단 메뉴 + 2열 또는 1열 자동 변환
- 모바일: 한 줄 큰 카드 + 하단 고정 메뉴

## 포함 기능
- 홈
- 내 서재
- 성경 연구
- 설교/기도문 편집
- 자동 저장
- 최근 작업 이어하기
- 이름 변경
- 큰 글씨
- 자료 백업

## 적용
GitHub 저장소 최상단의 아래 3개 파일을 교체하세요.
- index.html
- styles.css
- app.js

README.md는 설명 파일이라 선택입니다.
