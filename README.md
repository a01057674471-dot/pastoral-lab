# 목회 연구실 AI — 화성 아가페교회 황순영 목사님 전용

모바일·태블릿·PC에서 작동하는 반응형 웹 프로토타입입니다.

## 실행
`index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.

## 포함된 화면
- 맞춤형 홈 대시보드
- 성경 본문 연구: 본문·원어·배경·주석·비교
- AI 작업 도우미: 설교 정리·교회 글쓰기·설교 이미지·요청 다듬기
- 연구 노트와 로컬 자동 저장
- AI 교회 로드맵
- 모바일/태블릿/PC 반응형 UI

## 현재 단계
UI/UX 프로토타입이며, 실제 성경 데이터와 AI API는 연결 전입니다.

## 다음 개발 단계
1. OpenAI API 또는 선택한 AI 모델 연결
2. 성경 본문·원어·Strong 데이터 연동
3. 사용자 로그인 및 클라우드 저장
4. PDF/문서 내보내기
5. 교회 로드맵 문서 자동 생성
